# CiviSure - Public Safety Platform

## Features
- Anonymous Crime Reporting
- SOS Emergency Alerts
- Crime Zone Mapping
- Legal AI Chatbot
- Admin Dashboard

## Installation

1. Install dependencies:
```bash
npm install
```

2. Initialize database:
```bash
npm run init-db
```

3. Start server:
```bash
npm start
```

4. Open browser: http://localhost:3000
